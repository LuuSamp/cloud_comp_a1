Token totals come from CloudWatch in the Bedrock account (last 7 days). Chat requests and tool calls are counted in DynamoDB on the lab agent and reset if the sessions table is torn down. Set AGENT_USAGE_BUDGET_TOKENS and AGENT_USAGE_DAILY_BUDGET_TOKENS in .env.agent to show progress bars. The Bedrock IAM user needs cloudwatch:GetMetricData. Redeploy the agent after changing budgets.
